# 📊 Sales Dashboard — E-Commerce Analytics in Excel

> **A comprehensive Excel-based sales analytics project analyzing 51,291 e-commerce orders across multiple regions, product categories, and customer segments.**

---

## 📌 Project Overview

This repository contains two Excel dashboards built to analyze a large-scale global e-commerce sales dataset. The project demonstrates end-to-end data analytics — from raw data exploration to interactive visual dashboards.

---

## 📁 Files in This Repository

| File | Description |
|------|-------------|
| `E_Commerce_Dashboard_New.xlsx` | E-Commerce dashboard with sales analysis and visual charts |
| `Dynamic_Sales_Dashboard.xlsx` | Dynamic dashboard with pivot tables, histogram, and descriptive statistics |

---

## 📊 Dataset Description

**Source:** Global E-Commerce Sales Dataset
**Records:** 51,291 orders
**Period:** 2015 onwards

### Columns:

| Column | Description |
|--------|-------------|
| Order ID | Unique order identifier |
| Order Date | Date order was placed |
| Ship Date | Date order was shipped |
| Aging | Days between order and delivery |
| Ship Mode | Shipping method (First Class, Standard, etc.) |
| Product Category | Category of product sold |
| Product | Product name |
| Sales | Revenue from order ($) |
| Quantity | Number of items ordered |
| Discount | Discount applied (%) |
| Profit | Profit from order ($) |
| Shipping Cost | Cost of shipping ($) |
| Order Priority | Priority level (Critical, High, Medium, Low) |
| Customer Segment | Consumer, Corporate, Home Office |
| City / State / Country | Customer location |
| Region | Geographic region |
| Months | Month of order |

## 🛠️ Excel Skills Demonstrated

| Skill | Where Used |
|-------|-----------|
| **Pivot Tables** | Sales summary by category, region, segment |
| **Pivot Charts** | Visual breakdown of sales and profit |
| **Slicers** | Interactive filtering by category and region |
| **Dashboard Design** | KPI cards, charts, layout |
| **Descriptive Statistics** | Mean, median, mode, std dev of aging |
| **Histogram** | Distribution of order aging (delivery time) |
| **VLOOKUP** | Cross-referencing customer and product data |
| **Conditional Formatting** | Highlighting top performers |
| **Data Sorting & Filtering** | Exploring trends by region and category |

---

## 📈 Key Analysis Performed

### 1. Sales Performance
- Total sales, profit, and quantity by product category
- Monthly sales trends
- Regional performance comparison

### 2. Delivery Analysis
- Histogram of order aging (how long deliveries take)
- Descriptive statistics: mean delivery time, standard deviation
- Identifying delays by ship mode

### 3. Customer Analysis
- Sales breakdown by customer segment (Consumer, Corporate, Home Office)
- Top performing regions and cities

### 4. Product Analysis
- Best and worst performing product categories
- Profit margin analysis by category
- Discount impact on profit

## 💡 Key Insights

1. **Auto & Accessories** was among the top revenue-generating categories
2. **First Class** shipping had the highest order volume
3. **Consumer segment** accounted for the majority of orders
4. Orders with **higher discounts** showed lower profit margins
5. **Central region** had the highest number of orders globally

5. Explore **Pivot Table** sheet for detailed breakdowns
